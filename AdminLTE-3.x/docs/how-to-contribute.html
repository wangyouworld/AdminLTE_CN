为 AdminLTE 做贡献

始终欢迎你分享，也推荐你这么做！这是初学者的方法：开源入门请单击此处

  1. 贡献要求：
    • 当你提交时，你同意给 AdminLTE.io 一个非独占许可。我们（AdminLTE.io）可以任意使用该贡献。
    • 如果您使用第三方提供的内容，则必须使用开源许可证对其进行适当许可。
    • 仅通过 Github 拉取请求接受贡献。
    • 最后，贡献的代码必须适用于所有支持的浏览器（请参阅上面的浏览器支持）。
  2. 安装：
    • Fork 存储库（这里是指南）。
    • 克隆到你的电脑上
     git clone https://github.com/YOUR_USERNAME/AdminLTE.git
    • 创建新分支
  3. 编译 dist 文件（开发）：
    • 要编译 dist 文件，你需要 Node.js 10或更高版本/npm（node 包管理器）
    • 删除 ./package-lock.json 文件
    • npm install (install npm deps)
    • npm run dev （开发者模式，自动编译，支持 browsersync 实时演示）
    • 允许在 ./buid 目录、package.json 或 ./dist/js/demo.js 中修改，或在 HTML 文件中进行更改
    • 不要修改 ./dist/css/ 和 ./dist/js/ 目录下的文件，因为它是已编译文件
    • npm run production （在创建拉取请求之前，编译 css/js 文件并测试每个页面是否正常工作）
  4. 创建拉取请求

在线一键设置

你可以使用 Gitpod（一个开源的在线 IDE）或处理问题或管理 Prs。只需点击一下，就会自动启动工作区：

  • 克隆 AdminLTE 仓库
  • 安装依赖关系
  • 执行 npm run dev 启动服务

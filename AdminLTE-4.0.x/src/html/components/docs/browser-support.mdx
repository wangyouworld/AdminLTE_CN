AdminLTE 支持与 Bootstrap 5 相同的浏览器。

> ##### 摘自 Bootstrap 官方文档！
>
> Bootstrap 支持所有主流浏览器的**最新稳定版本**。
>
[查看更多信息](https://getbootstrap.com/docs/5.0/getting-started/browsers-devices/#supported-browsers)。

在 [.browserslistrc](https://github.com/ColorlibHQ/AdminLTE/blob/master/.browserslistrc) 文件中找到我们支持的浏览器及其版本：

```
# https://github.com/browserslist/browserslist#readme

>= 0.5%
最近 2 个主要版本
不再支持的浏览器除外
Chrome >= 60
Firefox >= 60
Firefox ESR
iOS >= 12
Safari >= 12
不支持 IE <= 11
```
